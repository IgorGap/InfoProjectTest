# ManagerCompanyReact

Для запуска в браузере необходимо:

Открыть терминал

Необходимо скачать зависимости командой
- npm install

Запустить командой 
- npm start
На сайте можно выбрать "Мои Компании" или  "Мои Менеджеры"
